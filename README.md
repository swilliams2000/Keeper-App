# Keeper App

A lightweight note-keeping application built with React that allows users to create and manage notes. This project focuses on component-based architecture, state management, and clean UI structure using modern frontend tooling.

## Features
- Create and display notes dynamically
- Reusable React components
- State management for user input and note rendering
- Clean and minimal UI styling

---

## Tech Stack
- **Frontend:** React
- **Build Tool:** Vite
- **Styling:** CSS
- **Hosting:** GitHub Pages

---

## Project Focus
This project was built to practice:
- React component structure
- Managing state with hooks
- Passing data between components
- Deploying a static React app with Vite and GitHub Pages
